  <action>
1. In `core/firewall.py`, **remove `_run_ps()` entirely**. Replace all PowerShell calls with direct `netsh` commands via list-form `_run()`:
   - `block_ip` Windows: `_run("netsh", "advfirewall", "firewall", "add", "rule", f"name={name}", "dir=in", "action=block", f"remoteip={ip}")` 
   - `unblock_ip` Windows: `_run("netsh", "advfirewall", "firewall", "delete", "rule", f"name={name}")`
   - `block_port` Windows: `_run("netsh", "advfirewall", "firewall", "add", "rule", f"name={name}", "dir=in", "action=block", f"protocol={proto}", f"localport={port}")`
   - `unblock_port` Windows: same delete pattern
   - `list_rules` Windows: `_run("netsh", "advfirewall", "firewall", "show", "rule", f"name={_RULES_PREFIX}", "verbose")` then parse output

2. Add IP validation gate at the top of `block_ip`, `unblock_ip`: call `is_valid_ip(ip)` from validators. If invalid, return `{"success": False, "message": "Invalid IP address"}` immediately.

3. Add port validation gate at the top of `block_port`, `unblock_port`: call `is_valid_port(port)`. If invalid, return failure.

4. In `utils/validators.py`, add `is_valid_ipv6(ip: str) -> bool` that checks for `isinstance(ipaddress.ip_address(ip), ipaddress.IPv6Address)`.

5. Update `tests/test_firewall.py` — add tests for injection rejection, IPv6, and valid calls.

6. Update `tests/test_validators.py` — add tests for `is_valid_ipv6()` and edge cases (empty string, None-like).
