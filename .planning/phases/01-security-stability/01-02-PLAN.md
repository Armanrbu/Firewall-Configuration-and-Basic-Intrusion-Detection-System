  <action>
1. Replace the global `_con` singleton with `threading.local()` storage:
   ```python
   import threading
   _local = threading.local()
   _DB_PATH = "firewall_ids.db"

   def get_db(path: str | None = None) -> sqlite3.Connection:
       """Return a per-thread SQLite connection (lazily initialized)."""
       db_path = path or _DB_PATH
       con = getattr(_local, 'connection', None)
       if con is None:
           con = _init_db(db_path)
           _local.connection = con
       return con
   ```

2. Add `close_db()` to close the current thread's connection:
   ```python
   def close_db() -> None:
       con = getattr(_local, 'connection', None)
       if con is not None:
           con.close()
           _local.connection = None
   ```

   - `stop()`: set event, `self._thread.join(timeout=5)`
   - `_loop()`: use `self._stop_event.wait(1)` instead of `time.sleep(1)` — allows instant wake on stop

2. **core/ids.py** — Add stop signaling to `IDSWorker`:
   - Add `_stop_requested` flag
   - Add `request_stop()` method
   - In the monitoring loop, check `_stop_requested` before each iteration
   - Ensure `run()` method exits cleanly when stopped

3. **ui/main_window.py** — Override `closeEvent`:
   ```python
   def closeEvent(self, event):
       self._shutdown()
       event.accept()

   def _shutdown(self):
       # Stop IDS worker
       if self._ids_worker:
           self._ids_worker.request_stop()
       if self._ids_thread:
           self._ids_thread.quit()
           self._ids_thread.wait(5000)
       # Stop scheduler
       from core.scheduler import get_scheduler
       get_scheduler().stop()
       # Stop API server
       if self._api_server:
           # Flask dev server — just let daemon thread die
           pass
       # Close all DB connections
       from core.blocklist import close_all_connections
       close_all_connections()
       logger.info("Application shutdown complete.")
   ```

4. **main.py** — Add `atexit` handler and `signal` handler for SIGINT/SIGTERM:

   ```python
   import atexit
   import signal
   def _cleanup():
       from core.scheduler import get_scheduler
       get_scheduler().stop()
       from core.blocklist import close_all_connections
       close_all_connections()
   atexit.register(_cleanup)
   ```
