<objective>
Create an abstract FirewallBackend interface with platform-specific implementations.

Purpose: The current `core/firewall.py` is a monolithic module mixing platform detection (`if _OS == "Windows"`) with business logic. This prevents clean testing, adding new platforms (nftables, WFP), and mocking in tests. The Strategy + Factory pattern from the architecture research gives us a clean ABC.

Output: `FirewallBackend` ABC, `WindowsNetshBackend`, `LinuxIptablesBackend`, `get_firewall_backend()` factory, refactored `core/firewall.py` as a thin facade.
</objective>

1. Create `core/firewall_abc.py`:

   ```python
   """Abstract firewall backend interface."""
   from abc import ABC, abstractmethod
   from typing import Any, TypedDict

   class FirewallResult(TypedDict, total=False):
       success: bool
       message: str
       rule_name: str
       rules: list
       profiles: dict

   class FirewallBackend(ABC):
       """Abstract interface for platform-specific firewall operations."""
       
       @abstractmethod
       def block_ip(self, ip: str, rule_name: str) -> FirewallResult: ...
       
       @abstractmethod
       def unblock_ip(self, ip: str, rule_name: str) -> FirewallResult: ...
       
       @abstractmethod
       def block_port(self, port: int, protocol: str, rule_name: str) -> FirewallResult: ...
       
       @abstractmethod
       def unblock_port(self, port: int, protocol: str, rule_name: str) -> FirewallResult: ...
       
       @abstractmethod
       def list_rules(self, prefix: str) -> FirewallResult: ...
       
       @abstractmethod
       def get_status(self) -> FirewallResult: ...
       
       @abstractmethod
       def enable(self) -> FirewallResult: ...
       
       @abstractmethod
       def disable(self) -> FirewallResult: ...
       
       @abstractmethod
       def enable_logging(self, log_path: str) -> FirewallResult: ...
   ```

2. Note: Backend methods receive pre-validated, pre-named arguments. Input validation stays in the facade (`core/firewall.py`).

1. Add `get_firewall_backend()` factory function to `core/firewall.py`:
   ```python
   _backend: FirewallBackend | None = None

   def get_firewall_backend() -> FirewallBackend:
       global _backend
       if _backend is None:
           if _OS == "Windows":
               from core.firewall_windows import WindowsNetshBackend
               _backend = WindowsNetshBackend()
           elif _OS == "Linux":
               from core.firewall_linux import LinuxIptablesBackend
               _backend = LinuxIptablesBackend()
           else:
               raise RuntimeError(f"Unsupported OS: {_OS}")
       return _backend

   def set_firewall_backend(backend: FirewallBackend) -> None:
       """Allow tests and plugins to inject a custom backend."""
       global _backend
       _backend = backend
   ```

2. Refactor all public functions to delegate to the backend:

   ```python
   def block_ip(ip: str, rule_name: str | None = None) -> dict[str, Any]:

       name = rule_name or _rule_name(ip)
       result = get_firewall_backend().block_ip(ip, name)
       logger.info("block_ip %s → %s", ip, result.get("message"))
       return result
   ```
