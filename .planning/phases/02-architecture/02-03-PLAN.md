<objective>
Replace raw sqlite3 with SQLAlchemy ORM and repository pattern.

Purpose: The current `core/blocklist.py` uses raw SQL strings with `sqlite3` directly. This works but: (1) makes switching to PostgreSQL impossible, (2) offers no type safety on query results, (3) mixes data access with business logic. SQLAlchemy ORM with a repository pattern solves all three.

Output: SQLAlchemy models for all tables, repository classes for CRUD operations, configurable backend (SQLite default, PostgreSQL optional), and updated callers.
</objective>

1. Create `core/models.py`:
   ```python
   """SQLAlchemy ORM models for NetGuard IDS."""
   from sqlalchemy import Column, Integer, Float, Text, String, create_engine
   from sqlalchemy.orm import DeclarativeBase, Session

   class Base(DeclarativeBase):
       pass

   class BlockedIP(Base):
       __tablename__ = "blocked_ips"
       ip = Column(String, primary_key=True)
       reason = Column(Text, default="")
       blocked_at = Column(Float)
       unblocked_at = Column(Float, nullable=True)
       auto_blocked = Column(Integer, default=0)
       expires_at = Column(Float, nullable=True)
       status = Column(String, default="active")

   class Alert(Base):
       __tablename__ = "alerts"
       id = Column(Integer, primary_key=True, autoincrement=True)
       ip = Column(String)
       type = Column(String)
       details = Column(Text, default="")
       timestamp = Column(Float)
       resolved = Column(Integer, default=0)
       ml_score = Column(Float, default=0.0)
       rule_id = Column(String, default="")
       action = Column(String, default="logged")

   class ConnectionLog(Base):
       __tablename__ = "connection_log"
       id = Column(Integer, primary_key=True, autoincrement=True)
       ip = Column(String)
       port = Column(Integer)
       protocol = Column(String)
       direction = Column(String)
       timestamp = Column(Float)
       details_json = Column(Text, nullable=True)

   class GeoCache(Base):
       __tablename__ = "geo_cache"
       ip = Column(String, primary_key=True)
       data_json = Column(Text)
       cached_at = Column(Float)

   class SchemaVersion(Base):
       __tablename__ = "schema_version"
       version = Column(Integer, primary_key=True)

   class ScheduledRule(Base):
       __tablename__ = "scheduled_rules"
       rule_id = Column(Integer, primary_key=True)
       data_json = Column(Text)
   ```
